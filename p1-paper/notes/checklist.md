# Submission Checklist

## Manuscript

- [ ] Title, authors, affiliations, and author running head finalized
- [ ] Abstract rewritten from placeholder into final camera-ready form
- [ ] Contribution list made explicit in the introduction
- [ ] All section placeholders replaced with prose or removed
- [ ] Page limit checked for the target venue
- [ ] Appendix inclusion reviewed against venue policy

## Evaluation

- [ ] Dataset A counts finalized
- [ ] Refactoring catalog frozen
- [ ] B1/B2/B3 implementations documented
- [ ] Timeout and cost budgets reported
- [ ] Confidence intervals regenerated for final tables and figures
- [ ] Failure-mode taxonomy internally consistent with results section

## Reproducibility

- [ ] Artifact URL or anonymized package inserted
- [ ] Toolchain versions pinned
- [ ] Replay command verified on a clean machine
- [ ] Figure and table sources preserved
- [ ] License information recorded for seed programs

## Submission Readiness

- [ ] Double-blind constraints checked if required by venue
- [ ] PDF metadata scrubbed if anonymization is needed
- [ ] All citations resolve against `../references.bib`
- [ ] All figures and tables are referenced in text
- [ ] Spellcheck and final read-through completed
